Imagre attempts to reconstruct images from fMRI data using CNNs and DCNNs and GANs

fmri

Functional Magnetic Resonance Imaging 
The fMRI measures the hemodynamic response that reflects the level of brain activity.  

Wikipedia

Encoder

Pre-Trained on VGG-19

We build the encoder network using VGG-19 model pre-trained on ILSVRC2012.  

Decoder

The fMRI decoder is constructed using a linear regression model. We use Ridge Regression with L2 normalization.  

Reconstruction Network

DCNN-GAN

The Reconstruction Network is a deconvolution network with 4 layers. The GAN is then trained using the output from the previous layers.  The GAN can optimize the generation of images by introducing category specific prior knowledge.
